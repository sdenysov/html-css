Сделать таблицу, которая растягивается на всю ширину экрана. 
Для этого придется для самой таблицы, тега html и для тега body 
задать высоту 100% и дать им маргин и паддинг 0.

Вся верстка состоит из трех таблиц:
	таблица №1 - самая основная таблица. Она состоит из трех строк в каждой из которых 
		по 6 ячеек.
		- В первой строке все 6 ячеек объеденены по горизонтали (значение colspan)
		- Во второй строке обычные ячейки
		- В третей строке с помощью того же значения colspan объеденены 
		  первая со второй и третья с четвертой, пятой и шестой. 

	таблица №2 - 5 строк в каждой из которых по одной ячейке
	таблица №3 - 3 строки в кадждой из которых по 2 ячейки. 
	
Обрати внимание на следующие свойства CSS, которые тебе пригодятся в разметке
	1 - vertical-align
	2 - text-align
	3 - Тегами table, th, tr и td
	4 - свойству для таблиц colspan
	5 - для некоторых колонок возможно придется задавать значение ширины.

Все td должны иметь padding 10px.

Задание не такое сложное как может показаться на первый взгляд.
